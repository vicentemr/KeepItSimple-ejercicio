1. Abri el archivo "index.html" en el navegador o ingresar a la ruta: https://vicentemr.github.io/KeepItSimple-ejercicio/
2. Seleccionar la opción "Choose file" y cargar el archivo Excel a importar.
3. Presionar el boton "Generar tabla".
4. Se creará la tabla con los datos del excel (agrupados por la columna "Supervisor" y ordenado alfabéticamente en cada grupo).
5. Se puede filtrar por: Supervisor, Clase, Departamento, Subsidiaria. ingresando un valor y seleccionando el botón "Filtrar".
6. Seleccionar el tipo de exportación (CSV, EXCEL, PDF) y se descargará su archivo.
